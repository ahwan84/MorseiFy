Quick Tips

  • • Dot (.) represents a short signal.
  • • Dash (-) represents a long signal.
  • • Separate letters using a single space.
  • • Separate words using / .
  • • Morse code ignores uppercase and lowercase letters.
  • • Practice by translating small words like SOS, HELP and HI.
