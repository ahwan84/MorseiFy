About MorseiFy

A simple and interactive Morse Code translator built using HTML, CSS and JavaScript.

About the Project

MorseiFy is a web application that converts English text into International Morse Code and vice versa. It also visually transmits Morse Code using a blinking light, helping users understand how Morse communication actually works.

Features

  • ✅ English → Morse Code conversion
  • ✅ Morse Code → English conversion
  • ✅ Supports alphabets, numbers and symbols
  • ✅ Animated Morse light transmission
  • ✅ Copy output with one click
  • ✅ Responsive dark-themed interface

Tech Stack

HTML5
CSS3
JavaScript
Font Awesome
