# ⚡ MorseiFy

A modern, responsive web application that converts **English Text ↔ International Morse Code** with an animated light transmission simulator.

> Learn Morse Code, translate messages, and visualize real-time Morse signaling—all in one place.

## ⚙️ How It Works

### Text → Morse

1. Type your message.
2. Click **Convert**.
3. The translated Morse Code is displayed.
4. The signal is transmitted through the animated light.

### Morse → Text

1. Enter Morse Code.
2. Click **Convert**.
3. Instantly receive the translated English text.

## 💡 Morse Light Simulation

The application simulates International Morse timing using JavaScript's asynchronous programming.

- Dot (•) → Short Blink
- Dash (—) → Long Blink
- Automatic pauses between letters
- Automatic pauses between words
